DESCRIPCION HTML

HTML es un lenguaje de marcado utilizado para crear páginas web. A través de diversas etiquetas, los desarrolladores definen los diferentes elementos de una página, como títulos, párrafos, imágenes y enlaces. Sin HTML, los navegadores no sabrían cómo organizar ni mostrar el contenido correctamente.



Término Descripción
HTML Lenguaje de marcado para la creación de páginas web.
Elementos Las etiquetas y contenido que forman la estructura de una página.
Atributos Propiedades añadidas a las etiquetas para definir características.
Estructura Básica Consiste en <!DOCTYPE>, <html>, <head> y <body>.
